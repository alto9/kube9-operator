# GH Project Item Add

Use the provided script to add an issue to a GitHub Project.

## Usage

Run the script:

`node scripts/gh-project-item-add.js <issue-ref> [project-url]`

- If `project-url` is omitted, `.forge/project.json.github_board` is used.
